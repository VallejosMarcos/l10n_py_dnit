# Uso

## Impuestos

- **Ventas**: el IVA se calcula según la configuración del producto; por
  defecto IVA 10%.
- **Compras**: asignar el impuesto soportado en las facturas de proveedor para
  el cálculo del crédito fiscal.

## Plan de cuentas

- Consultar las cuentas en
  **Contabilidad → Configuración → Plan de cuentas**.
- Registrar asientos en
  **Contabilidad → Asientos contables** usando las cuentas del plan.

## Reportes

Con el plan cargado quedan disponibles los reportes estándar de Odoo: Balance
General, Estado de Resultado y reportes de impuestos (IVA).
